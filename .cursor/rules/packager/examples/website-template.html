Troubleshooting

Agent not showing up in Copilot CLI

Ensure the .agent.md file is in ~/.copilot/agents/ and has valid YAML frontmatter with name and description fields. Restart the Copilot CLI session.

Skill not auto-loading

Skills load based on relevance to your prompt. Try mentioning keywords from the skill's description. You can also check loaded skills with /skills.

Templates not found

Verify the template files exist in ~/.copilot/skills/{name}/templates/. The skill references templates by relative path from its own directory.
